## Lancement de l'application


### Installation / Lancement de l'app 


1. Cloner le projet
   ```sh
   git clone https://github.com/Soluce-Development/Maquette
   ```
2. installation des librairies
   ```sh
    pip install -r requirement.txt
   ```
4. Lancer le projet `main.py`
   ```py
   python main.py
   ```
